# How to run

1. You can Sign up or Login using two different Py File

2. To SignUP:   Run Calling_SignUp_Methods.py file
3. To Login:    Run Calling_Login_Methods.py file
   Both files first ask to enter PlatformName and UDID from user to set desired capabilities.
   There User need to write appropriate PlatformName and UDID.
   
4. You can see locators (Paths of different android view elements) Open Locators.py file to view all locators used in the `project.
5. You can see all methods which are used to set desired capabilities and also used for Login and SignUp.  Open Testing_Methods.py to view.
